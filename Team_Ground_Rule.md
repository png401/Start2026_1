# 📢 Team Ground Rules

## 1. Communication (소통)

| 규칙 | 설명 |
|-----|-----|
| 응답 시간 | 카카오톡 및 팀 채팅 메시지는 **6시간 이내 응답**을 원칙으로 한다. |
| 일정 공유 | 불참 또는 지연이 생길 경우 **최소 24시간 전 공지**한다. |
| 채널 구분 | **카카오톡은 잡담**, **디스코드는 업무 관련 소통**으로 구분하여 사용한다. |
| 질문 문화 | 모르는 부분이 있으면 혼자 고민하지 말고 **적극적으로 질문한다.** |
| 태도 | 감정적 발언을 자제하고 **문제 중심으로 대화**한다. |
| 기본 매너 | **안읽씹 금지**, 빠른 응답을 습관화한다. |

---

## 2. Meeting (회의)

| 규칙 | 설명 |
|-----|-----|
| 정기 회의 | **정기 회의 일정 : 매주 금요일 저녁 9시** |
| 회의 방식 | **회의 방식 : 비대면 - 디스코드 활용** |
| 회의 준비 | 회의 시작 전 **회의록을 미리 읽고 참석**한다. |
| 회의 태도 | 다른 사람의 발언을 **끊지 않고 모든 의견을 들은 후 피드백**한다. |
| 회의 기록 | 회의록은 **팀장이 작성 후 공유**한다. |
| 아이디어 피드백 | 아이디어 제안 시 **칭찬 → 의견 제시** 순서로 피드백한다. |
| 진행 방식 | 회의는 **화면 공유 기반으로 진행**한다. |

---

## 3. Project Workflow (프로젝트 진행)

| 규칙 | 설명 |
|-----|-----|
| 역할 관리 | 각 역할과 담당 업무는 **Notion 및 Discord에 명시하고 주기적으로 업데이트**한다. |
| 협업 | 다른 팀원의 역할도 이해하고 프로젝트 전체 흐름을 함께 파악한다. |
| 문제 공유 | 어려운 사항이 있으면 **숨기지 말고 솔직하게 공유**한다. |
| AI 사용 | AI의 도움은 받되, AI의 답변을 이해 없이 그대로 사용하지 않는다. |
| 태도 | 프로젝트에 **책임감과 애정을 가지고 참여**한다. |

---

## 4. Conflict Resolution (갈등 해결)

| 규칙 | 설명 |
|-----|-----|
| 의사결정 | 의견 충돌 시 **다수결이 아닌 충분한 토론 후 결정**한다. |
| 존중 | 최종 결정 사항은 **팀원 모두가 존중한다.** |

---

## 5. Study & 기타

| 규칙 | 설명 |
|-----|-----|
| 공부 기록 | 개인 공부 일지를 **Notion에 업데이트**한다. |
| 공지 확인 | **공지사항 확인 시간 : 상시** |
